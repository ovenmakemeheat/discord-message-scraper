# ก๊ายไก่ (Chicky Chatbot)

## Overview

This channel tracks the development of "Chicky" — an AI chatbot for querying broiler chicken Work Instructions (WI) and standard compliance documents. The project uses a RAG (Retrieval-Augmented Generation) approach with vector stores and LLM API calls.

## Key Technical Details

- **Architecture**: RAG pipeline — documents indexed into a vector database, queries answered via LLM API calls
- **Platform**: Google Cloud Platform (Vertex AI Workbench, Data Store)
- **GCP Project**: `aiml-cpf-th-farm-dev`
- **GPU Instance**: T4, instance name "penguin"
- **Challenges**: Slow vectorstore retrieval on local (~25s per query), merged-cell table parsing, role naming conventions
- **Data**: WI (Work Instructions) for chicken farming, farm certification/standard requirement documents
